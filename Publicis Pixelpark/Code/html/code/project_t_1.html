Zeus

Dein Sprach - Assistent

Zeus ist ein Programm mit automatischer Spracherkennung und kann deinen Computer rundum automatisieren. Er kann Programme öffnen, schließen, minimieren und vieles mehr. Auch hat Zeus eine anbindung im Netz, sodas du ihn mit seiner App auch über das Internet steuern kannst, alles über Sprachbefehle. Leise und sicher läuft Zeus im Hintergrund und wartet auf einen Sprachbefehl, die man sich auch anzeigen lassen kann.
